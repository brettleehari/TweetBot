## 🎮 **Quick Start**

```bash
# Clone the repository
git clone https://github.com/brettleehari/TweetBot.git
cd TweetBot

# Install dependencies
npm install

# See agents in action! 🚀
npm run demo-core-agency

# Check system status
npm run agent-cli status

# Query agent decisions
npm run agent-cli query market-hunter
```

## 🧠 **Technical Architecture**

### **Core Agency Framework**
- `GoalHierarchy`: Self-modifying objective structures
- `ReputationModel`: Performance-based trust scoring  
- `MarketRegime`: Environmental condition detection
- `AgentPersonality`: Behavioral trait modeling

### **Agent Decision Cycle**
1. **Environmental Assessment**: Market condition analysis
2. **Goal Evaluation**: Objective priority reassessment  
3. **Strategic Planning**: Action selection with confidence scoring
4. **Decision Execution**: Autonomous action with reasoning
5. **Outcome Learning**: Performance feedback and adaptation

### **Database Schema**
- `agent_suggestions`: All agent decisions and reasoning
- `strategic_decisions`: High-level system optimizations
- `alpha_discoveries`: Market opportunities and patterns
- `feedback_logs`: Performance and adaptation tracking
